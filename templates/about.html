Our Background

What is PricePal? & What does it do?

PricePal is a web-based price comparison application designed to simplify online shopping by aggregating product prices across multiple e-commerce platforms. The system empowers users to make informed purchasing decisions by presenting real-time pricing data and product availability from various online retailers in one unified interface. Developed using HTML, CSS, and JavaScript for the frontend, the backend is powered by Python with Flask, while MySQL is used for efficient data storage and retrieval.

Web scraping techniques are employed to dynamically fetch and update product data from different sources. PricePal enhances the user experience by offering features such as keyword-based search, product filtering, and price trend visualization. The application aims to provide transparency in online pricing and optimize the shopping experience by helping users find the best deals available across the web.
